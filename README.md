## 🚀 Quick start

- Install

  ```bash
  make install
  # or
  npm install
  ```

- Run all challenges

  ```bash
  make run
  ```

- Run specific challenge

  ```bash
  make run YEAR=YYYY CHALLENGE=XX # where YYYY is the year and XX is the challenge number
  ```
